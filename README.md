AI-powered image editor with **9+ tools** — grayscale, cartoonify, edge detection, super resolution, and more.

## 🚀 Features

| Feature | Description |
|---------|-------------|
| 🎨 **Grayscale** | Convert images to black and white |
| 🖌️ **Sepia** | Vintage-style filter |
| 🔍 **Blur** | Gaussian blur effect |
| ✨ **Sharpen** | Enhance image details |
| 📐 **Edge Detection** | Outline image edges |
| 🎭 **Cartoonify** | Transform photos into cartoons |
| 🌅 **Brightness/Contrast** | Adjust lighting and contrast |
| 📈 **Super Resolution** | Upscale images by 2x |
| 🖼️ **Side-by-Side Preview** | Compare original and edited |

## 🛠️ Tech Stack

- **Frontend:** Streamlit
- **Image Processing:** OpenCV, PIL (Pillow)
- **Deployment:** Streamlit Cloud
- **Language:** Python 3.11+

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/Aikaksh-Singh-Routela/ai-image-editor.git
cd ai-image-editor

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run app.py
